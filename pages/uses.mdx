With the time I have used many different technologies to work. This is a list of my current preferred stack.

## Tools

- [Visual Studio Code](https://code.visualstudio.com/) as my main IDE.
- [Brave](https://brave.com) as my main browser.
- [Prettier](https://prettier.io) (with a VSCode plugin) with the default configuration.
- [Hyper](https://hyper.is/) as my terminal.
- [Postman](https://www.getpostman.com/) to try API endpoints.
- [GitHub](https://github.com) for my code repositories.

## Tech Stack

- [TypeScript](https://www.typescriptlang.org/) as my main programming language.
- [React.js](https://reactjs.org) for UI creation.
- [Tailwind](https://tailwindcss.com/) for styling.
- [SWR](https://swr.now.sh/) for remote data fetching.
- [Next.js](https://nextjs.org/) as my Frontend framework.
- [Jest](https://jestjs.io/) for unit testing.
- [React Testing Library](https://testing-library.com/react) for React testing.
- [Cypress](https://www.cypress.io/) for integration and E2E testing.
- [Contentz](https://github.com/sergiodxa/contentz) as my Pure Static Site Generator.
- [Netlify](https://netlify.com/) to deploy my blog.
- [ZEIT Now](https://zeit.co/now) to deploy any Next.js application.
- [GitHub Action](https://github.com/features/actions) for continuous integration, delivery and deployment pipelines.
