Sobre mí

Hola! Soy Sergio, tengo 22 años y vivo en Buenos Aires, Argentina. Soy Desarrollador Front End y JavaScript con algunos años de experiencia. Trabajé un tiempo en una companía llamada Promaker haciendo diseños con HTML5 y CSS3 y programando JS en un pequeño equipo.

Estudié eletromecánica en la escuela secundaria E.T. N°11 D.E. 6 "Manuel Belgrano" y cuando me gradué (2011) hice los cursos de Diseño Web, FrontEnd Profesional, Node.js y JavaScript Profesional y Git y Github Profesional de Platzi.

Me gusta compartir código en mi cuenta de Github y escribir artículos y tutoriales que publico en Medium y en Cristalab. Hice algunas librerías para JavaScript y LESS que uso en mi trabajo y libero con licencia MIT en Github.

Me gusta usar herramientas que me ayuden en mi trabajo, como LESS como pre-procesador de CSS y Gulp para automatizar mis tareas repetitivas.
